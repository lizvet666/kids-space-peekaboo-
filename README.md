# いないいないばあ（キッズスペース用）

1〜2歳向けの、カーテンをタップするとキャラが「ばあ」と出てくるWebアプリです。複数箇所に隠れているので、何人かで同時に別々の場所をタップして遊べます。

## 使い方

- **タップ／クリック**: 色付きのカーテンをタップすると、その場所だけカーテンが開き、キャラが「ばあ」と出てきます。効果音が鳴ります。
- **複数人**: 6箇所にキャラが隠れているので、同時に別々の場所をタップできます。順番待ちは不要です。
- 約2.5秒後にカーテンが閉まり、またタップして遊べます。

---

## 運用方法

### 提案・デモ時（オンライン）

クライアントにスマホやPCで見せるときは、**オンラインでURLを共有**すると便利です。

1. このフォルダを **Git** で管理し、**GitHub** にプッシュ
2. **GitHub Pages** または **Render** で静的サイトとしてデプロイ
3. 発行されたURLをクライアントに送り、「このリンクを開いてください」と案内

同じURLをどこからでも開けるので、提案時に楽です。

### 本番運用（オフライン）

キッズスペースのディスプレイではオフラインで使う場合の手順です。

1. **このフォルダ一式**（`index.html`・`style.css`・`script.js`）を、運用するPCやタブレットにコピー（USBやクラウドで）
2. その端末で `index.html` をブラウザで開く  
   - または `npx serve .` などで簡易サーバーを立て、ブラウザで `http://localhost:3000` を開く
3. ディスプレイをキッズモード／フルスクリーンで表示

音は Web Audio API で鳴らしているため、オフラインのままでも効果音が再生されます。

---

## ファイル構成

- `index.html` … メイン画面・6箇所の隠れスポット
- `style.css` … フルスクリーン・グリッド・カーテンとキャラのスタイル・開閉アニメーション
- `script.js` … タップ検知・「ばあ」表示・効果音・自動で閉じる処理

## 技術メモ

- 静的ファイルのみ。サーバーサイド不要
- キャラは絵文字（🐱🐶🐻🐰🐼🦊）で表示。画像に差し替える場合は `.character` のスタイルを変更可能
